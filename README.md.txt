<h2>Para configurar uma redundância de tráfego de dados usando dois switches e dois cabos</h2> <p>Você pode configurar uma tecnologia de agregação de link chamada Link Aggregation Control Protocol (LACP). O LACP permite que você configure vários links físicos em um único link lógico, aumentando a largura de banda e fornecendo redundância.</p> <p>Aqui está um exemplo de como configurar a redundância de tráfego de dados usando dois switches e dois cabos :</p> <ol> <li>Conecte um cabo de rede do primeiro switch à porta 1 e outro cabo de rede à porta 2.</li> <li>Conecte um cabo de rede do segundo switch à porta 1 e outro cabo de rede à porta 2.</li> <li>No primeiro switch, configure as portas 1 e 2 para formar um LACP com o segundo switch. Use o seguinte comando:</li> < /ol> <pre><code>Switch1(config)# interface range gigabitEthernet 0/1 - 2 Switch1(config-if-range)# channel-group 1 mode active </code></pre> <ol start=" 4"> <li>No segundo switch, configure as portas 1 e 2 para formar um LACP com o primeiro switch. Use o seguinte comando:</li> </ol> <pre><code>Switch2(config)# interface range gigabitEthernet 0/1 - 2 Switch2(config-if-range)# channel-group 1 mode active </code ></pre> <p>Verifique se o link está funcionando corretamente usando o seguinte comando:</p> <pre><code>Show running-config </code></pre> <p>Com essas configurações, você terá dois cabos que funcionam juntos para aumentar a largura de banda e fornecer redundância.
